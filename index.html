Introducing My Site

I'm an aspiring web developer living on the high desert plains of eastern Colorado. My goal is to create modern, fast, responsive, and accessible cross-platform progressive web applications.

My ideal stack doesn't have a cool acronym like MEAN. I'm not sure it's technically a stack. I like using React to scaffold my applications (create-react-app), MaterializeCSS for UI/UX and styling, and Google Firebase for backend stuff like the database or cloud functions. I use Firebase for hosting as well because it's awesome. I can write, run, test, and deploy from the CLI in VS Code - which is also awesome.
